# pebblearn-api
## Description
[Nest](https://github.com/nestjs/nest) API server for pebblearn learning platform application.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
